OvirtMetrics is an ActiveRecord-based gem for reading the oVirt History database.

## Installation

Add this line to your application's Gemfile:

    gem 'ovirt_metrics'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ovirt_metrics
